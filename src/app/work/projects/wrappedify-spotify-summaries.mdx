## Overview

Developed an Android app that offers users an engaging way to interact with their music data year-round, rather than waiting for annual summaries.

## Key Features

- **Spotify API Integration**: Leveraged the Spotify Web API to generate on-demand, personalized summaries of user listening habits.
- **Social Engagement**: Integrated **Firebase** to allow users to post summaries on a social media-style feed, fostering community interaction.
- **Secure Infrastructure**: Utilized Firebase for secure user data storage, account setup, and authentication.

## Technologies Used

- **Java & Kotlin**: For native Android development.
- **Firebase**: For backend infrastructure and real time social features.
- **Android Studio**: Primary IDE for development and debugging.

## Outcome

Created a functional, shareable music experience that provides users with immediate insights into their streaming preferences through an interactive interface.
